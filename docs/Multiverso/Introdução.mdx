# Introdução

O servidor utiliza um plugin que permite a criação de mundos paralelos. Esse tutorial por sua vez, ensina como utilizar-se desses mundos, e o motivo da existencia de mundos paralelos.
Sobre o motivo, os mundos tem objetivos especificos, listados na documentação de cada mundo.

# Tutorial

Por incrivel que pareça, todo o trabalho para utilizar esses mundos, é apenas 1 comando :D.

Utilize o comando `/worldtp` para se teleportar para um mundo paralelo.

Os mundos paralelos disponiveis estão listados no resto dessa documentação.
