Aqui estão listados recomendações de uso para o servidor, e seus respectivos links.

## Cliente para Minecraft Original

Recomendamos o uso do Multimc ou Polymc para os jogadores que tenham uma conta original do minecraft. É mais pratico e permite a instalação facilitada de todas as ferramentas abaixo.

MultiMC: https://fpt.icu/multimc

## Cliente para Minecraft Pirata

Não recomendamos a utilização do Minecraft pirata, mas caso você seja esse jogador, utilize um Cliente limpo e sem virus, recomendamos o UltimMC, ele é identico ao Multimc para conta Original, e te permite seguir os tutoriais aqui com facilidade:

Linux: https://fpt.icu/UltimMCL<br />
Windows: https://fpt.icu/UltimMCW

## Customização

Para computadores mais fracos, recomendamos instalar um modpack no cliente usando Fabric, o Fabulously Optimized, ele permite otimizar o jogo para não lagar em batatas (sim kkkk), mas caso seu computador tenha mais alguns cavalos de poder de sobra, recomendamos o uso da textura Faithful com a shader Complementary, tudo isso é opcional, mas pode deixar seu jogo tanto mais rapido, como mais bonito.

Modpack (Fabric) recomendado: https://fpt.icu/fabuoptm<br />
Textura recomendada: https://fpt.icu/faithful<br />
Shaders recomendadas: https://fpt.icu/compShaders
